## Project Description

Vapulus Backend Challenge

chat application that enables the customers to sign up, login, and join a general chat room, send messages, show previous messages of the chat.


## Project Dependencies

| Dependencies  | Version         | 
| :------------ |:---------------:| 
| Node.js      | >= 12.13.1       |
| Typescript      | >= 4.3.5      |
| MongoDB      | >= 4.2.1         | 
| @nestjs/cli | >= 8.0.0          |


## Installation

```bash
$ npm install
$ mv .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
